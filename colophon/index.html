Colophon

Authored by Dan Burzo, and released under the MIT license.

Culori is the Romanian word for ‘colors’. The logo is typeset in Hatch by Mark Caneso.

Culori is indebted to Mike Bostock's D3.js and Gregor Aisch's chroma.js. D3, in particular, has been a treasure-trove of ideas and academic references.

The code is formatted with prettier and bundled with rollup, buble, and terser. Tests use tape. The library does not have external run-time dependencies.

This website is statically generated using Eleventy by Zach Leatherman.
